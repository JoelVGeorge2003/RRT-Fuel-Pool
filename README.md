The file contains a seperate folder called access codes
open linux terminal and open ros2

the **launch robot** terminal code will help to access the uuv.urdf file and open the robot geometry in rviz
the **combined cloud** and **voxel split** terminal commands will help to open the point cloud map and then create the voxels to be seperated from it which is dose_cloud_combined_csv_pub.py and dose_cloud_to_voxel_box_split.py
the **automation** terminal code will run the rrtstar_nav_6dof.py and publish the path required 
the goal setting terminal command can be used to set the goal position in the combined pool.
